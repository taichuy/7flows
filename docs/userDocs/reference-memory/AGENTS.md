# reference-memory

本目录用于存放引用记忆，不记录正文结论，只记录“去哪里看什么”。

## 分类说明

- `source-reference.md`
  - 源码参考类引用。
  - 例如：`../dify`、`../xyflow` 这类可作为设计、交互、实现参考的源码目录。
- `api-reference.md`
  - API 查看相关引用。
  - 例如：本项目 OpenAPI、第三方接口文档、调试入口地址。
- `script-reference.md`
  - 脚本相关引用。
  - 例如：初始化脚本、构建脚本、运维脚本、数据处理脚本的入口说明。

新增引用文档时，优先归入以上分类；若后续出现稳定的新类别，再新增独立文档。
